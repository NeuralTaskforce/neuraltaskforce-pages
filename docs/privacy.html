Privacy Policy

What this application is

NeuralTaskForce is a personal automation tool built and operated by a single developer (cheito.groove) to manage and publish music video content to social media platforms including TikTok and YouTube. It is not a public product or service.

Data collection

This application does not collect, store, or process personal data from any user other than its sole developer. No analytics, tracking, or telemetry of any kind is implemented.

Third-party platform access

The application connects to the following platforms on behalf of the developer only:

  • TikTok — via TikTok Content Posting API, to upload video content to the developer's own account (@cheito.groove)
  • YouTube — via YouTube Data API v3, to upload video content to the developer's own channel

Access tokens used to authenticate with these platforms are stored solely on the developer's private infrastructure and are never shared with third parties.

Data sharing

No information is sold, shared, or disclosed to any third party. The application does not transmit any data beyond what is strictly necessary to interact with the APIs listed above.

Contact

For any questions regarding this privacy policy, contact the developer via TikTok (@cheito.groove).
