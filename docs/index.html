NeuralTaskForce

Personal content automation platform for independent music creators.

About

NeuralTaskForce is a private workflow automation tool built for a single creator — cheito.groove, an independent bass musician. It automates the publishing of video content to social media platforms including TikTok and YouTube.

The application is not a public service or SaaS product. It is used exclusively by its developer for personal content management.

What it does

  • Uploads bass cover videos and short-form highlights to TikTok and YouTube
  • Automates video metadata (titles, descriptions, hashtags)
  • Generates short highlight clips from longer recordings using FFmpeg
  • Manages the full publishing workflow from a single automation engine (n8n)

TikTok API Usage

This application uses the TikTok Content Posting API exclusively to upload video content to the creator's own TikTok account (@cheito.groove). No third-party accounts are accessed. No user data is collected or stored.

Scopes requested:

video.upload video.publish
